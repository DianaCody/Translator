# Eclipse翻译小助手
------
###***Eclipse翻译插件***
#####Eclipse翻译插件，调用有道翻译API，支持在线实时查询单词、取词翻译。
#####插件项添加在工具栏里。
------

插件工具特点：

> * 直接在Eclipse IDE里即可得到翻译取词，无需另外再开启单独词典功能
> * 代码支持同步翻译
> * 实时查询，方便快捷
